The OpenGolfSim Core Simulator is free golf simulator software and training tool with hyper-realistic ball flight physics. It uses shot data from a compatible [launch monitor](/desktop/shot-data/launch-monitor/) as the input.

{: .note }
> OpenGolfSim Core is still in early beta. Join our [Discord](/connect-with-us/) to get the most up-to-date info.


Features:

- Easy and free to get use
- Direct launch monitor integrations (Approach R10, Square)
- Built with the latest Unity 6 engine
- Integrated swing video, control, and shot data
- Developer friendly APIs
