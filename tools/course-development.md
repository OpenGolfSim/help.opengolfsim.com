### Surfaces Tags (optional)

You can tag any of your terrains or meshes as specific surface types. Just select the game object and select a tag.

Make sure the tags in your Unity project match the following: 

- `Fairway` (Tag 0)
- `Water` (Tag 1)
- `Green` (Tag 2)
- `Rough` (Tag 3)
- `DeepRough` (Tag 4)
- `SemiRough` (Tag 5)
- `Fringe` (Tag 6)
