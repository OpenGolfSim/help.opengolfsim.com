## Built-In Buttons

- Aim / Distance
- Club Up / Club Down
- Mulligan
- Reset Aim
- Flyover
- Toggle Map
- Toggle Putting
- Heatmap
- Drop
- Re-Hit
- Score Card
- Range Finder
